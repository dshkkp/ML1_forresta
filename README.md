
# Machine Learning 

Проект по основам машинного обучения и первичному анализу данных.

## Описание

В проекте разбираются базовые этапы построения ML-модели:

1. сбор и загрузка данных;
2. анализ признаков и целевой переменной;
3. обработка выбросов;
4. визуализация данных;
5. создание признаков;
6. обучение и сравнение моделей.

Основная задача проекта — предсказать цену аренды квартиры по данным объявления.
